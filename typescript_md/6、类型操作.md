## 泛型类

```js
泛型类具有与泛型接口相似的形状,泛型类在尖括号(<>) 跟在类名后面
                        
                        
class test <t>{
    name:t;
    add:(a:t)=>t
}
let t=new test<number>()
t.name=1
t.add=function(a){
    return a
}
```

## typeof 类型操作符 ----有点不明白

```js
1 typeof 的常规操作

2 预定义类型 ReturnType
ReturnType<t> 预定义类型，有一个泛型，泛型类型必须是一个有返回值的函数
type xx=(参数:类型)=>类型
ReturnType 就是返回值函数返回值的
```

## 索引访问类型

```js
索引类型本身就是一个类型
索引访问类型可以是一个联合类型

keyof 与索引访问类型
type xx=xxx[typeof xxx]
将会得到一个xx是一个联合类型，由xxx索引组成的联合类型

```
